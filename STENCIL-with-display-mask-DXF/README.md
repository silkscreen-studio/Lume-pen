To get the stencil for soldering the FPC and detach the Display light mask coupon, the stencil can be ordered from JLCPCB with the GERBER file provided here. There is also a .dxf file just in case.

This stencil has all the pads aperture for the FPC, as well as 5 detachable coupons, 100 microns thin (select 0.1mm thickness) steel plate that shapes the segments from the LEDs on the display. The coupons should detach easily by cutting the tabs with a cutter.

The stencil coupon is also available in STEP.
