To get the stencil for soldering the FPC and detach the Display light mask coupon, the stencil can be ordered from JLCPCB with the GERBER file provided here. There is also a .dxf file just in case.

This stencil has all the pads aperture for the FPC, as well as 5 detachable coupons, 100 microns thin steel plate that shapes the segments from the LEDs on the display. The coupons should detach easily by cutting the tabs with a cutter.

JLCPCB order: select 0.1mm thickness, custom size 100x100. Should be $3 (unless additionnal fees for the irregular shape, we'll see how it turns out).

The stencil coupon is also available in STEP.
